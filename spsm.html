About the project

This project was part of a course called Design for userexperience. We worked toards a company called SPSM. They are a authority that's dedicated to helping kids with cognitive disabilities. They where developing a "new" application that was supposed to gather several applications into one. Our task was to try to create a prototype of this, implementing all applications into one while still kepping it user friendly and especially easy to learn. They emphesised the easy to learn part since a lot of people that are going to use the application are older men and women.
